List of properties used

  1. height: Sets the height of an element
  2. padding: Adds space within an element between the content and its border
  3. margin: Sets space outside of an element
  4. background: Sets the background color/image of an element
  5. font-family: Sets the font typeface of text
  6. display: Specifies how an element should be displayed
  7. flex-direction: Sets the direction of flex items within a flex container
  8. justify-content: Aligns flex items along the main axis of a flex container
  9. position: Sets the position of an element
  10. top: Sets the top position of an absolutely positioned element
  11. left: Sets the left position of an absolutely positioned element
  12. right: Sets the right position of an absolutely positioned element
  13. bottom: Sets the bottom position of an absolutely positioned element
  14. box-shadow: Adds a shadow effect to an element
  15. z-index: Sets the stack order of an element
  16. border-radius: Sets the rounded corner of an element's border
  17. align-items: Aligns flex items along the cross axis of a flex container
  18. min-width: Sets the minimum width of an element
  19. color: Sets the color of text
  20. cursor: Specifies the type of cursor to be displayed when hovering over an element
  21. font-weight: Sets the weight (thickness) of text
  22. font-size: Sets the size of text
  23. transition: Adds a transition effect to an element's property and can define time of it
  24. opacity: Sets the transparency of an element
  25. transform: Applies a transformation to an element
  26. text-shadow: Adds a shadow effect to text
  27. text-decoration: Adds decoration to text
  28. column-gap: Sets the gap between columns in a multi-column layout
  29. row-gap: Sets the gap between rows in a multi-column layout
  30. max-width: Sets the maximum width of an element
  31. outline: Draws a line around an element
  32. outline-offset: Sets the space between an outline and the border or edge of an element
  33. line-height: Sets the vertical spacing between lines
  34. and more...
